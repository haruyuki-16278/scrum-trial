# スク・ラム・トライアル (Scrum Trial) 機能概要書

## 1. はじめに
本アプリケーション「Scrum Trial」は、スクラム開発のプロセスを楽しみながら体験・学習できる、Webベースのシミュレーションゲームです。
参加者は開発チームの一員となり、リアルタイムで他のメンバーと協力しながら、タスクの消化、スプリントの運営、そして振り返り（KPT）を通じてプロジェクトの進行を体験します。

## 2. 主な機能

### 2.1 リアルタイム・コラボレーション
複数人のユーザーが同時に同じセッション（プロジェクトルーム）に参加し、状態を共有します。
誰かがタスクを移動したり、ダイスを振ったりすると、即座に全員の画面に反映されます。

### 2.2 スクラムイベントのシミュレーション
スクラムの主要なイベントをフェーズとして実装しています。
*   **プランニング**: プロダクトバックログ（PBL）の作成と、スプリントバックログへのタスク選定。
*   **スプリント**: ダイスロールによる日々のタスク消化シミュレーション（Day 1〜Day 4）。
*   **レビュー/レトロスペクティブ**: スプリントの成果確認と、KPT法を用いた振り返り。

### 2.3 ゲーミフィケーション（ダイスシステム）
タスクの進捗は「ダイス（サイコロ）」の出目によって決まります。
*   メンバーは1スプリント中の各「日（Day）」に1回ダイスを振ることができます。
*   出目の合計がタスクの見積もり工数（12ポイント）を超えると、タスクは「完了（Done）」となります。
*   スクラムマスター（SM）などの役割に応じた特殊なアクションやボーナス要素も存在します。

### 2.4 共有ノート（KPTエディタ）
マークダウン記法をサポートしたリッチテキストエディタを搭載。
レトロスペクティブ時のKPT（Keep, Problem, Try）の記録などに使用でき、複数人で同時に編集が可能です（Google Docsのような使用感）。

## 3. ユーザー体験フロー

### ステップ 1: セッションの開始
*   **作成**: ユーザーは任意の名前で新しいセッション（部屋）を作成します。
*   **参加**: 生成されたセッションIDを共有することで、他のメンバーが同じ部屋に参加できます。
*   **再開**: ブラウザの履歴機能により、以前参加していたセッションに再接続できます。

### ステップ 2: プランニング (Planning)
*   **タスク作成**: 「プロダクトバックログ」列に新しいタスクを追加します。
*   **タスク選定**: 今回のスプリントで取り組むタスクを「スプリントバックログ」列にドラッグ＆ドロップで移動します。
*   **スプリント開始**: 準備ができたらフェーズを進め、スプリントを開始します。

### ステップ 3: スプリント (Sprint Day 1-4)
*   **担当決定**: 自分が取り組むタスクを選択（Click）します。
*   **作業実施**: 「Roll Dice」ボタンをクリックしてダイスを振ります。出目に応じてタスクの進捗バーが伸びます。
*   **完了**: 進捗が100%（12pt）に達すると、タスクは「Done」列に自動的に移動します。
*   **日次進行**: 全員がアクションを終えると、次の「日（Day）」に進みます。4日経過するとスプリント終了です。

### ステップ 4: 振り返り (Retrospective)
*   **KPT実施**: 画面右側の共有ノートを使って、今回のスプリントの良かった点、悪かった点、次への改善策を話し合いながら記録します。
*   **プロセス改善**: スクラムマスターはプロセス改善のダイスを振り、成功すると次回のスプリントでボーナスダイス（ダイス数増加）を獲得できます。

### ステップ 5: 次のスプリントへ
*   振り返りが終わると、再びプランニングフェーズに戻ります。
*   残ったタスクや新たなタスクをもとに、次のスプリント計画を立てます。
*   全3スプリント（設定により変更可能）を完走するとゲーム終了（Finished）です。

## 4. 画面構成
メイン画面（ワークスペース）は、スクラムボードを模した4カラム構成になっています。
1.  **Product Backlog**: 全ての要求事項（タスク）のリスト。
2.  **Sprint Backlog**: 現在のスプリントで実施中のタスク一覧。
3.  **Dice & Team**: チームメンバーの状況確認、ダイスロールの実行、フェーズ操作パネル。
4.  **Shared Notes**: リアルタイム共有エディタ（KPT用）。
