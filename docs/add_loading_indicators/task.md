# ローディング・スケルトン表示の追加

## 概要
ユーザーの操作（ロール切り替え、タスク追加、タスク移動など）に対して、処理中であることを示す視覚的なフィードバック（ローディング表示やスケルトン表示）を追加する。
現状では操作後に即座に反応がなく、処理が実行されているかが分かりづらいため、これを改善する。

## 要件
1.  **タスク追加（プロダクトバックログ）**:
    *   追加ボタンを押した後、完了するまでボタンを無効化し、ローディングインジケータを表示する。
2.  **タスク移動（プロダクトバックログ ↔ スプリントバックログ）**:
    *   移動ボタンを押した後、完了するまでボタンを無効化し、ローディングインジケータを表示する。
3.  **ロール切り替え（チームパネル）**:
    *   ロールアイコンをクリックした後、反映されるまでアイコン上にローディングインジケータを表示する。
4.  **その他**:
    *   必要に応じてスケルトン表示を導入するが、まずは上記アクションに対するフィードバックを優先する。
